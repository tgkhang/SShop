It for Cloudfront signed URLs. You need to create a key pair and add the public key to AWS. Then you can use the private key to sign your URLs.

create private key - do not push that to git

```bash
openssl genrsa -out private_key.pem 2048
```

create public key

```bash
openssl rsa -pubout -in private_key.pem -out public_key.pem
```

add your public key to aws key and create key group
fix the access in cloudfront distribution
